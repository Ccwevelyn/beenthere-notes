###### 梯度下降算法
目标：使损失函数找到最小值

方式：梯度下降

+ 函数的梯度（**gradient**）指出了函数的**最陡**增长方向。梯度方向走，函数增长得就越快。
+ 按照梯度的**负方向**走，函数值自然就降低的最快了。
+ 模型训练的目标是寻找合适的 w 与 b 以最小化代价函数值。

#### CNN 卷积神经网络（Convolutional neural network）
 Designed to process grid-like data such as images.  

| **Input Layer** | Receives raw image data (e.g., 32×32×3 for RGB). | 输入层接收原始图像数据，保持空间结构。 |
| --- | --- | --- |
| **Convolutional ****Layer** | Applies filters (kernels) to** extract features** like edges and textures. | 卷积层通过卷积核**提取局部特征**，如边缘、纹理。 |
| **Activation ****Layer** | Introduces **non-linearity (**ReLU, Tanh). | 激活层引入**非线性**，使模型能学习复杂模式。 |
| **Pooling ****Layer** | Reduces spatial dimensions (Max Pooling, Average Pooling). | 池化层**降低特征图尺寸**，减少计算量并防止过拟合。 |
| **Flattening** | Converts feature maps into a vector. | 展平层**将多维特征图转为一维向量**。 |
| **Fully ****Connected ****Layer** | Performs classification based on extracted features. | 全连接层根据特征进行最终分类。 |


<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/42556572/1777269045016-dca52db2-7f6d-476f-8bbe-e14703ad906b.png)

1. **连接方式**
+ 普通神经网络：每个神经元与上一层所有输入相连 → 参数量随输入维度急剧增加。
+ CNN：每个卷积核只与输入的一小块区域相连 → 参数量大幅减少。
2. **权重使用**
+ 普通神经网络：每个连接都有独立权重。
+ CNN：同一个卷积核的**权重**在整个输入空间**共享**，保证模型能识别相同模式（如边缘）在不同位置出现。
    - 平移不变性** translation invariance **平移不影响当前特征
3. **求和内容**
+ 普通神经网络：对所有输入特征做加权求和。
+ CNN：对局部区域做加权求和（卷积），再滑动窗口重复计算。

###### 卷积处理多通道信息
先局部，再整体

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/42556572/1777269936843-50ba5700-b10f-4acb-9692-e5b8980ab1c2.png)

1. **局部处理 (Local processing)：**卷积核只处理输入的一小块区域（例如 3×3 的局部区域）。
2. **跨通道加权求和 (Channel-wise summation)：**对于多通道输入（如 RGB），卷积核在每个通道的局部区域分别加权求和。
3. **整体输出 (Global aggregation) **把所有通道的结果相加，得到该位置的一个整体输出值。
4. **滑动生成特征图 (Sliding to form feature map)：**卷积核在输入上滑动，重复上述过程，形成完整的特征图。

###### 卷积的设计原理
1. 参数量
    - 为什么神经网络处理图片效果不好？--> 参数过多
    - CNN 相比全连接网络**大幅减少参数量**。它不是为每个像素都分配独立权重，而是用小卷积核，并在不同位置共享权重 Convolution uses small kernels with** shared weights. ** 
        * 连线数目由 convolution kernel 决定
2. 局部相关性 Local Correlation  图像具有强烈的局部相关性——相邻像素之间存在联系。卷积通过处理局部区域来利用这一特性，提取边缘、纹理和形状。  
3. 人类的视觉习惯 Human Visual Habit： 人类视觉也是先局部处理，再整体整合。CNN 模仿这一习惯，在浅层识别简单模式，在深层识别复杂对象。

###### 卷积的特征提取
<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2026/png/42556572/1777271500509-50fca589-2c83-4fa7-8cbb-541faf5f4641.png)

在 CNN 中，“多层”是它最重要的设计思想之一。每一层都在不同层次上提取特征，逐步从简单到复杂。

This multi-layer hierarchy mimics human vision: local details first, then global understanding.

+ **浅层特征 (Shallow features)**
    - ****Early convolutional layers capture low-level features such as edges, lines, and simple textures.
    - 比如边缘、线条和简单纹理。
+ **中层特征 (Intermediate features)**
    - Middle layers **combine** these simple features into more complex patterns, such as corners, shapes, or repeated textures. 中层卷积层把简单特征组合成更复杂的模式，比如角点、形状或重复纹理。
+ **深层特征 (Deep features)**
    - Deeper layers capture high-level semantic features 高级语义特征, such as faces, objects, or categories.

在感受野一定的情况下，层数越多，每一层的输出就是上一层特征的组合。这样层层叠加，提取到的特征就更加丰富和显著。  

+ **感受野**：决定单个神经元能看到的输入范围。
+ **多层结构**：在固定感受野下，后续层不是扩大范围，而是综合前层的特征。
+ **结果**：多层让模型从简单特征逐步组合成复杂模式，即使感受野大小不变，特征表达也更强。

###### 池化操作
池化（Pooling）是 CNN 中的重要操作，用来**缩小特征图的尺寸，同时保留主要信息**。它的核心思想是：在一个小区域内进行聚合（最大值或平均值），减少数据量但保留关键特征。

+ **Max Pooling**
    - 在局部区域取最大值，突出最显著的特征。
+ **Average Pooling**
    - 在局部区域取平均值，使特征更平滑。

**作用**

+ Reduce computation and parameters
+ Prevent overfitting
+ Provide translation invariance 提供平移不变性（同一特征在不同位置仍能识别）

举例

+ 输入特征图大小：4×4
+ 使用 2×2 的 Max Pooling → 每个区域取最大值 → 输出特征图大小变为 2×2。
