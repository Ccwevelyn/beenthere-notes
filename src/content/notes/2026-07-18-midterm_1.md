###### <font style="color:rgb(0,0,0);">Benefits of data preprocessing (Ch2)</font>
+ **Accuracy**: Ensures data is accurate and free of manual entry errors.
+ **Completeness**: Handles missing values to ensure data is complete
+ **Consistency:** Maintains uniformity in data across different sources.
+ **Timeliness**: Ensures data is updated regularly.
+ **Trustability**: Verifies data comes from trustworthy sources.
+ **Interpretability**: Converts raw data into a format that can be understood and used.



###### <font style="color:rgb(0,0,0);">Patterns and strategies for missing data (Ch2)</font>
+ **Deletion**
    - **Listwise Deletion**
    - **Pairwise Deletion**
+ **Imputation**
    - **Mean/Median Imputation**
    - **Forward/Backward Fill**
    - **Predictive Imputation**
+ **Choosing the right approach **
    - **MCAR -> deleting rows or columns **
    - **MAR -> imputation of data**
    - **MNAR -> improve dataset; find data**

| **Mean/Median ****Imputation** | 均值或中位数填补 | 简单易用   可能扭曲数据分布，低估方差 |
| --- | --- | --- |
| **Forward/Backward ****Fill** | 用前一个或后一个**非缺失值填补** | 适用于时间序列   <u>不适合剧烈波动数据</u> |
| **Predictive ****Imputation** | 用**其他变量建模预测**缺失值 | 更准确，保留变量间关系   计算复杂，需假设模型合理性 |


| **MCAR** | Missing Completely At Random | Listwise / Pairwise Deletion | **缺失与任何变量无关**，删除不会引入偏差 |
| --- | --- | --- | --- |
| **MAR** | Missing At Random | Imputation    | 缺失与**其他变量**有关，可用这些变量估计缺失值 |
| **MNAR** | Missing Not At Random | Improve dataset,<br/>find data | 缺失与自身值相关，无法仅靠现有数据推断，需额外信息或重新收集 |

**R 中的六种基本数据类型（Data Types）**

1. **Numeric（数值型）** 表示所有数字，包括整数和小数。
2. **Character（字符型）** 表示文本或字符串, 用于存储名称、标签、描述等非数值信息。
3. **Integer（整数型）** 表示没有小数部分的整数。与 numeric 不同，integer 更节省内存，适合计数类变量。
4. **Logical（逻辑型）** 表示布尔值：`TRUE` 或 `FALSE`。
5. **Factor（因子型）** 表示分类变量，有固定的类别集合。适合处理分类数据。
6. **Complex（复数型）** 表示复数，包含实部和虚部。

使用 `class()` 函数可以查看变量的数据类型。

**R 中的类型转换函数（Type Conversion Functions）**

+ `as.character(x)`：将对象转换为字符型
+ `as.numeric(x)`：转换为数值型（可包含小数）
+ `as.integer(x)`：转换为整数型（去除小数部分）
+ `as.logical(x)`：转换为逻辑型（非零数值会变成 `TRUE`）
+ `as.factor(x)`：转换为因子型（分类变量）
