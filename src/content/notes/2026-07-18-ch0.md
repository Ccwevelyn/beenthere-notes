#### Model related to Big data
Computing model -  决定了计算资源如何被使用、数据如何被处理，以及程序如何运行。

**<font style="color:rgb(0, 29, 53);">Big data </font>**<font style="color:rgb(0, 29, 53);">refers to </font>enormous, complex, and rapidly growing datasets that traditional data processing software cannot handle

<u>对“大数据”发展非常关键的计算模型</u>：

+ **<font style="color:rgb(0,0,0);">Distributed computing	</font>**<font style="color:rgb(0,0,0);">-  Processes and manages algorithms across many machines in a computing environment</font>
    - **分布式计算**：指的是将一个计算任务拆分后，分配到多个计算机上同时运行。
    -  让多个机器协同处理数据，提升处理速度和系统稳定性  
+  **Virtualization** - The creation of a virtual (rather than actual) version of computing resources, such as an operating system, a server, a storage device, or network resources  
    -  是指用软件技术把真实的硬件资源“虚拟”成多个独立的资源单元  
    -  虚拟机  
    -  提高资源利用率，灵活部署和管理计算环境

##### Data mining
+ **<u>Definition</u>**：The process of analyzing data to **extract information** not offered by the raw data alone 
    - <font style="color:rgb(39, 50, 57);">Data mining is the process of extracting useful and previously </font>**<font style="color:rgb(39, 50, 57);">unknown patterns</font>**<font style="color:rgb(39, 50, 57);"> from large datasets</font>
    - <font style="color:rgb(39, 50, 57);">Transform raw data into meaningful and understandable information</font>
    - 通过算法、统计方法或机器学习，**<font style="background-color:#FBDE28;">挖掘出隐藏的模式、趋势或洞察</font>**。  
+ **<u>3 main elements of data mining</u>****：**
    -  Data  
    -  Discovery - <font style="color:rgb(0,0,0);">identifying	</font>**<font style="color:rgb(0,0,0);">new	</font>**<font style="color:rgb(0,0,0);">patterns, trends ,and insights</font>
    -  Deployment - <font style="color:rgb(0,0,0);">implementing discoveries  把发现的结果应用到实际业务  </font>

+ **<u>Data Mining ANALYSIS TECHNIQUES</u>**

Data Mining **分析技术**：

<!-- 这是一张图片，ocr 内容为： -->
![](https://cdn.nlark.com/yuque/0/2025/png/42556572/1756729321486-c2f75702-b02f-4e72-bd44-a15b450b167f.png)

1. **Data Profilling** （数据剖析）

对现有数据源中的**统计数据和信息**进行**收集**的过程

分析已有数据源的结构、质量和分布。  

how many? average?

清洗数据前了解字段缺失率、异常值、重复项等  

2. **Data replication**（数据复制）

 在**多个数据源**之间同步数据，确保**一致性。** 

3. **Recommendation engine**（推荐引擎）

 分析用户行为，推荐相关产品或内容。  



这些技术不是直接用于挖掘数据模式，而是用于**支撑数据挖掘的环境和应用场景**：

| 技术 | 作用 | 分类定位 |
| --- | --- | --- |
| **Data profiling** | 检查数据质量、统计特征 | 数据**准备**阶段 |
| **Data replication** | 保证多个数据源一致性 | **数据**工程/系统维护 |
| **Recommendation engine** | 用挖掘结果驱动应用（如推荐） | 应用层算法（非通用分析方法） |


📘 这些技术更偏向“**数据挖掘的外围支持**”，或者“**挖掘结果的应用"**

 经典的数据挖掘算法（建模与模式识别）**建模技术**：

1. **Classification**（分类）

**<font style="color:rgb(0,0,0);">Assigns</font>**<font style="color:rgb(0,0,0);"> records to one of a</font><font style="color:rgb(0,0,0);background-color:#FBDE28;"> </font>**<font style="color:rgb(0,0,0);background-color:#FBDE28;">predefined set</font>**<font style="color:rgb(0,0,0);background-color:#FBDE28;"> </font><font style="color:rgb(0,0,0);">of classes</font>

2. **Estimation**（估计）

**<font style="color:rgb(0,0,0);">Determines values</font>**<font style="color:rgb(0,0,0);"> for an unknown </font><u><font style="color:rgb(0,0,0);">continuous</font></u><font style="color:rgb(0,0,0);"> variable behavior or </font>**<font style="color:rgb(0,0,0);">estimated	future value - </font>**<font style="color:rgb(0,0,0);">预测</font>**<font style="color:rgb(0,0,0);">连续变量</font>**<font style="color:rgb(0,0,0);">的值 </font>

3. **Affinity Grouping**（关联分组）

<font style="color:rgb(0,0,0);">Determines which things go together</font>

+  购物篮分析  
4. **Clustering**（聚类）

将数据划分为**若干相似子群体**。  

<font style="color:rgb(0,0,0);">Segments a heterogeneous population	of records into a number of more	 homogeneous	subgroups</font>

| <font style="color:rgb(0,0,0);">维度</font> | **<font style="color:rgb(0,0,0);">分类（Classification）</font>** | **<font style="color:rgb(0,0,0);">聚类（Clustering）</font>** |
| --- | --- | --- |
| <font style="color:rgb(0,0,0);">是否有标签</font> | ✅<font style="color:rgb(0,0,0);"> 有（监督学习）</font> | ❌<font style="color:rgb(0,0,0);"> 无（非监督学习）</font> |
| <font style="color:rgb(0,0,0);">输入数据</font> | **<font style="color:rgb(0,0,0);">已知类别</font>**<font style="color:rgb(0,0,0);">的数据样本</font> | **<font style="color:rgb(0,0,0);">未知类别</font>**<font style="color:rgb(0,0,0);">的数据样本</font> |
| <font style="color:rgb(0,0,0);">输出结果</font> | <font style="color:rgb(0,0,0);">为每个样本</font>**<font style="color:rgb(0,0,0);">分配</font>**<font style="color:rgb(0,0,0);">一个已有的</font>**<font style="color:rgb(0,0,0);">类别</font>** | <font style="color:rgb(0,0,0);">将样本</font>**<font style="color:rgb(0,0,0);">划分为</font>**<font style="color:rgb(0,0,0);">若干组（类别未知）</font> |
| <font style="color:rgb(0,0,0);">目标</font> | <font style="color:rgb(0,0,0);">学习如何预测标签</font> | <font style="color:rgb(0,0,0);">发现数据中的结构或相似性</font> |
| <font style="color:rgb(0,0,0);">应用场景</font> | <font style="color:rgb(0,0,0);">垃圾邮件识别、疾病诊断、图像分类</font> | <font style="color:rgb(0,0,0);">客户细分、市场分析、异常检测</font> |
